# F1 2025 Championship Finale Simulator

A fully interactive, single-file HTML application to simulate the conclusion of the 2025 Formula 1 season.

### 1. Interactive Drag & Drop Interface
Rearrange the grid for the Sprint and Main races. The **Live Standings** on the left update instantly as you move drivers, showing you exactly what each position change means for the title fight.

![Qatar Sprint Simulation](screenshots/sprint.png)

### 2. Live Points Calculation
Supports different point systems for Sprint races (top 8) and Grand Prix races (top 10). It visually indicates the "Points Cutoff" line.

![Live Standings Update](screenshots/race.png)

### 3. Championship Trajectory Analysis
After the final round, visualize the championship battle with a dynamic graph showing the points progression of the top three contenders across the final three sessions.

![Final Results Graph](screenshots/results.png)
